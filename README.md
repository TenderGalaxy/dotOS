# dotOS
A multithreaded modularized Operating System that simplifies codeloading, FS, and data uploading and downloading.


## Building the OS
1. Clone this repository:
```bash
git clone "https://www.github.com/tendergalaxy/dotOS"
```
2. Turn the puller into an executable:
```bash
chmod +x pull.sh
```
3. Build the repository into bloxd code:
```bash
./pull.sh
```
